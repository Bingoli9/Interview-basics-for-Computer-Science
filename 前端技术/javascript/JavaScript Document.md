## JavaScript Document

document 对象是文档的根节点，window.documen。 

浏览器载入HTML文档之后，document对象就开始存在，可以直接调用

document子节点类型 

- document.doctype，表示文档类型节点(DocumentType)
- document.documentelement，表示元素节点(element)
