#### 消息队列

消息队列加锁了，怎么做到无锁编程或者尽可能降低锁的影响

#### 锁优化

不得不加锁，那怎么优化锁呢。开源项目有很多优化例子，注意！！


#### mojo怎么用
mojom文件定义发送端和接收端通信接口  

发送端：  
- 继承mojom文件定义的接口
- 成员函数包括mojo::Binding\<T\>, T为mojom文件定义的接口  

接收端：  
- 继承mojom文件定义接口
- 实现通信接口，用于处理消息
