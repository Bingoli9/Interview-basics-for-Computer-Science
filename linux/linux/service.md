## service

Service command is used to manager the system services, such as start, stop, restart, check the status and so on.

Relatived command includes chkconfig, ntsysv and etc.

Command **service** is an linux shell script, and it will find specific service script, and will call the serivce's shell to finish the task.

```C++
service <service> start
service <service> stop
service <serviec> restart
```
