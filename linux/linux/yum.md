## yum

#### find software package

```C++
yum search
```

#### list all packages can be installed

```C++
yum list
```

#### list all packages can be updated

```C++
yum list updates
```

#### list all packages installled already

```C++
yum list installed
```

#### get packages info

```C++
yum info
```

#### list all files package provides

```C++
yum provides
```
